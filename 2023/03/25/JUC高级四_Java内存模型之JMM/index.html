JUCй«ҳзә§еӣӣ:JavaеҶ…еӯҳжЁЎеһӢд№ӢJMM

1. и®Ўз®—жңәзЎ¬д»¶еӯҳеӮЁдҪ“зі»(JMMдёәд»Җд№ҲиҜһз”ҹ)

еӣ дёәжңүиҝҷд№ҲеӨҡзә§зҡ„зј“еӯҳ(cpuе’Ңзү©зҗҶдё»еҶ…еӯҳзҡ„йҖҹеәҰдёҚдёҖиҮҙзҡ„)пјҢCPUзҡ„иҝҗиЎҢе№¶дёҚжҳҜзӣҙжҺҘж“ҚдҪңеҶ…еӯҳиҖҢжҳҜе…ҲжҠҠеҶ…еӯҳйҮҢиҫ№зҡ„ж•°жҚ®иҜ»еҲ°зј“еӯҳпјҢиҖҢеҶ…еӯҳзҡ„иҜ»е’ҢеҶҷж“ҚдҪңзҡ„ж—¶еҖҷе°ұдјҡйҖ жҲҗдёҚдёҖиҮҙзҡ„й—®йўҳ

еңЁжҲ‘д»¬cpuеҜ„еӯҳеҷЁе’ҢеҶ…еӯҳд№Ӣй—ҙеҠ дёҠй«ҳйҖҹзј“еӯҳжқҘе№іиЎЎеҜ„еӯҳеҷЁзҡ„еӯҳеӮЁйҖҹеәҰиҝңеӨ§дәҺеҶ…еӯҳзҡ„еӯҳеӮЁйҖҹеәҰзҡ„й—®йўҳ

JavaиҷҡжӢҹжңәи§„иҢғдёӯиҜ•еӣҫе®ҡд№үдёҖз§ҚJavaеҶ…еӯҳжЁЎеһӢпјҲjava Memory ModelпјҢз®Җз§°JMM) жқҘ==еұҸи”ҪжҺүеҗ„з§ҚзЎ¬д»¶е’Ңж“ҚдҪңзі»з»ҹзҡ„еҶ…еӯҳи®ҝй—®е·®ејӮпјҢ==
д»Ҙе®һзҺ°и®©JavaзЁӢеәҸеңЁеҗ„з§Қе№іеҸ°дёӢйғҪиғҪиҫҫеҲ°дёҖиҮҙзҡ„еҶ…еӯҳи®ҝй—®ж•ҲжһңгҖӮжҺЁеҜјеҮәжҲ‘д»¬йңҖиҰҒзҹҘйҒ“JMM

2. JavaеҶ…еӯҳжЁЎеһӢJava Memory Model

JMM(JavaеҶ…еӯҳжЁЎеһӢJava Memory ModelпјҢз®Җз§°JMM)жң¬иә«жҳҜдёҖз§ҚжҠҪиұЎзҡ„жҰӮеҝөе№¶дёҚзңҹе®һеӯҳеңЁе®ғ==д»…д»…жҸҸиҝ°зҡ„жҳҜдёҖз»„зәҰе®ҡжҲ–и§„иҢғ==пјҢйҖҡиҝҮиҝҷз»„и§„иҢғе®ҡд№үдәҶзЁӢеәҸдёӯ(е°Өе…¶жҳҜеӨҡзәҝзЁӢ)еҗ„дёӘеҸҳйҮҸзҡ„иҜ»еҶҷи®ҝй—®ж–№ејҸе№¶еҶіе®ҡдёҖдёӘзәҝзЁӢеҜ№е…ұдә«еҸҳйҮҸзҡ„еҶҷе…ҘдҪ•ж—¶д»ҘеҸҠеҰӮдҪ•еҸҳжҲҗеҜ№еҸҰдёҖдёӘзәҝзЁӢеҸҜи§ҒпјҢе…ій”®жҠҖжңҜзӮ№йғҪжҳҜ==еӣҙз»•еӨҡзәҝзЁӢзҡ„еҺҹеӯҗжҖ§гҖҒеҸҜи§ҒжҖ§е’ҢжңүеәҸжҖ§еұ•ејҖзҡ„гҖӮ==

2.1 иғҪе№Іеҳӣ?

  1. йҖҡиҝҮJMMжқҘе®һзҺ°зәҝзЁӢе’Ңдё»еҶ…еӯҳд№Ӣй—ҙзҡ„жҠҪиұЎе…ізі»гҖӮ
  2. еұҸи”Ҫеҗ„дёӘзЎ¬д»¶е№іеҸ°е’Ңж“ҚдҪңзі»з»ҹзҡ„еҶ…еӯҳи®ҝй—®е·®ејӮд»Ҙе®һзҺ°и®©JavaзЁӢеәҸеңЁеҗ„з§Қе№іеҸ°дёӢйғҪиғҪиҫҫеҲ°дёҖиҮҙзҡ„еҶ…еӯҳи®ҝй—®ж•ҲжһңгҖӮ

3.==JMMи§„иҢғдёӢдёүеӨ§зү№жҖ§==

3.1 еҸҜи§ҒжҖ§

жҳҜжҢҮеҪ“дёҖдёӘзәҝзЁӢдҝ®ж”№дәҶжҹҗдёҖдёӘе…ұдә«еҸҳйҮҸзҡ„еҖјпјҢе…¶д»–зәҝзЁӢжҳҜеҗҰиғҪеӨҹз«ӢеҚізҹҘйҒ“иҜҘеҸҳжӣҙ пјҢJMMи§„е®ҡдәҶжүҖжңүзҡ„еҸҳйҮҸйғҪеӯҳеӮЁеңЁдё»еҶ…еӯҳдёӯгҖӮ

3.1.1 зәҝзЁӢи„ҸиҜ»й—®йўҳ

Javaдёӯжҷ®йҖҡзҡ„е…ұдә«еҸҳйҮҸдёҚдҝқиҜҒеҸҜи§ҒжҖ§пјҢеӣ дёәж•°жҚ®дҝ®ж”№иў«еҶҷе…ҘеҶ…еӯҳзҡ„ж—¶жңәжҳҜдёҚзЎ®е®ҡзҡ„пјҢеӨҡзәҝзЁӢе№¶еҸ‘дёӢеҫҲеҸҜиғҪеҮәзҺ°вҖқи„ҸиҜ»вҖқпјҢжүҖд»ҘжҜҸдёӘзәҝзЁӢйғҪжңүиҮӘе·ұзҡ„е·ҘдҪңеҶ…еӯҳпјҢзәҝзЁӢиҮӘе·ұзҡ„е·ҘдҪңеҶ…еӯҳдёӯдҝқеӯҳдәҶиҜҘзәҝзЁӢдҪҝз”ЁеҲ°зҡ„еҸҳйҮҸзҡ„дё»еҶ…еӯҳеүҜжң¬жӢ·иҙқпјҢзәҝзЁӢеҜ№еҸҳйҮҸзҡ„жүҖжңүж“ҚдҪңпјҲиҜ»еҸ–пјҢиөӢеҖјзӯү пјүйғҪеҝ…йңҖеңЁзәҝзЁӢиҮӘе·ұзҡ„е·ҘдҪңеҶ…еӯҳдёӯиҝӣиЎҢпјҢиҖҢдёҚиғҪеӨҹзӣҙжҺҘиҜ»еҶҷдё»еҶ…еӯҳдёӯзҡ„еҸҳйҮҸгҖӮдёҚеҗҢзәҝзЁӢд№Ӣй—ҙд№ҹж— жі•зӣҙжҺҘи®ҝй—®еҜ№ж–№е·ҘдҪңеҶ…еӯҳдёӯзҡ„еҸҳйҮҸпјҢзәҝзЁӢй—ҙеҸҳйҮҸеҖјзҡ„дј йҖ’еқҮйңҖиҰҒйҖҡиҝҮдё»еҶ…еӯҳжқҘе®ҢжҲҗ

дёӘдәәжҖ»з»“:Javaдёӯжҷ®йҖҡзҡ„е…ұдә«еҸҳйҮҸдёҚдҝқиҜҒеҸҜи§ҒжҖ§,еҒҮи®ҫеңЁеӨҡзәҝзЁӢжғ…еҶөдёӢ,жҜҸдёӘзәҝзЁӢеҗҜеҠЁж—¶йғҪдјҡд»Һдё»еҶ…еӯҳжӢ·иҙқдёҖд»Ҫе…ұдә«еҸҳйҮҸеүҜжң¬еҲ°иҮӘе·ұзәҝзЁӢз§Ғжңүзҡ„еҶ…еӯҳдёӯ,дҪҶз”ұдәҺеҸҳйҮҸж•°жҚ®еңЁзәҝзЁӢд№Ӣй—ҙдёҚеҸҜи§Ғ,зәҝзЁӢдёҖдҝ®ж”№дәҶиҮӘе·ұзәҝзЁӢеҶ…зҡ„еүҜжң¬еҸҳйҮҸжҲ–иҖ…дё»еҶ…еӯҳдёӯзҡ„еҸҳйҮҸ,еҜ№зәҝзЁӢдәҢжҳҜдёҚеҸҜи§Ғзҡ„,зәҝзЁӢдәҢд»»з„¶дҪҝз”Ёзҡ„жҳҜиҮӘе·ұзәҝзЁӢзҡ„еүҜжң¬еҸҳйҮҸ

==жҲ‘д»¬жғіиҰҒзәҝзЁӢдёҖдҝ®ж”№иҮӘе·ұзәҝзЁӢеҶ…зҡ„еүҜжң¬еҸҳйҮҸжҲ–иҖ…дё»еҶ…еӯҳдёӯзҡ„еҸҳйҮҸж—¶и®©зәҝзЁӢдәҢеҸҜи§Ғиҝҷе°ұжҳҜеҸҜи§ҒжҖ§==

3.2 еҺҹеӯҗжҖ§

жҢҮдёҖдёӘж“ҚдҪңжҳҜдёҚеҸҜдёӯж–ӯзҡ„пјҢеҚіеӨҡзәҝзЁӢзҺҜеўғдёӢпјҢж“ҚдҪңдёҚиғҪиў«е…¶д»–зәҝзЁӢе№Іжү°

3.3 жңүеәҸжҖ§

3.3.1 жҢҮд»ӨйҮҚжҺ’

еҜ№дәҺдёҖдёӘзәҝзЁӢзҡ„жү§иЎҢд»Јз ҒиҖҢиЁҖпјҢжҲ‘д»¬жҖ»жҳҜд№ жғҜжҖ§и®Өдёәд»Јз Ғзҡ„жү§иЎҢжҖ»жҳҜд»ҺдёҠеҲ°дёӢпјҢжңүеәҸжү§иЎҢгҖӮдҪҶдёәдәҶжҸҗеҚҮжҖ§иғҪпјҢзј–иҜ‘еҷЁе’ҢеӨ„зҗҶеҷЁйҖҡеёёдјҡеҜ№жҢҮд»ӨеәҸеҲ—иҝӣиЎҢйҮҚж–°жҺ’еәҸгҖӮJavaи§„иҢғи§„е®ҡJVMзәҝзЁӢеҶ…йғЁз»ҙжҢҒйЎәеәҸеҢ–иҜӯд№үпјҢеҚіеҸӘиҰҒзЁӢеәҸзҡ„жңҖз»Ҳз»“жһңдёҺе®ғйЎәеәҸеҢ–жү§иЎҢзҡ„з»“жһңзӣёзӯүпјҢйӮЈд№ҲжҢҮд»Өзҡ„жү§иЎҢйЎәеәҸеҸҜд»ҘдёҺд»Јз ҒйЎәеәҸдёҚдёҖиҮҙ,жӯӨиҝҮзЁӢеҸ«жҢҮд»Өзҡ„йҮҚжҺ’еәҸгҖӮ

жҢҮд»ӨйҮҚжҺ’еҸҜд»ҘдҝқиҜҒдёІиЎҢиҜӯд№үдёҖиҮҙпјҢдҪҶжІЎжңүд№үеҠЎдҝқиҜҒеӨҡзәҝзЁӢй—ҙзҡ„иҜӯд№үд№ҹдёҖиҮҙпјҢеҚіеҸҜиғҪдә§з”ҹвҖқи„ҸиҜ»вҖқпјҢз®ҖеҚ•иҜҙ:дёӨиЎҢд»ҘдёҠдёҚзӣёе№Ізҡ„д»Јз ҒеңЁжү§иЎҢзҡ„ж—¶еҖҷжңүеҸҜиғҪе…Ҳжү§иЎҢзҡ„дёҚжҳҜз¬¬дёҖжқЎпјҢдёҚи§Ғеҫ—жҳҜд»ҺдёҠеҲ°дёӢйЎәеәҸжү§иЎҢпјҢжү§иЎҢйЎәеәҸдјҡиў«дјҳеҢ–гҖӮ

д»Һжәҗз ҒеҲ°жңҖз»Ҳжү§иЎҢзӨәдҫӢеӣҫ:

еҚ•зәҝзЁӢзҺҜеўғйҮҢйқўзЎ®дҝқзЁӢеәҸжңҖз»Ҳжү§иЎҢз»“жһңе’Ңд»Јз ҒйЎәеәҸжү§иЎҢзҡ„з»“жһңдёҖиҮҙгҖӮ
еӨ„зҗҶеҷЁеңЁиҝӣиЎҢйҮҚжҺ’еәҸж—¶еҝ…йЎ»иҰҒиҖғиҷ‘жҢҮд»Өд№Ӣй—ҙзҡ„ж•°жҚ®дҫқиө–жҖ§
еӨҡзәҝзЁӢзҺҜеўғдёӯзәҝзЁӢдәӨжӣҝжү§иЎҢ,з”ұдәҺзј–иҜ‘еҷЁдјҳеҢ–йҮҚжҺ’зҡ„еӯҳеңЁпјҢдёӨдёӘзәҝзЁӢдёӯдҪҝз”Ёзҡ„еҸҳйҮҸиғҪеҗҰдҝқиҜҒдёҖиҮҙжҖ§жҳҜж— жі•зЎ®е®ҡзҡ„,з»“жһңж— жі•йў„жөӢ,==жүҖд»Ҙжңүж—¶еҖҷжҲ‘д»¬дјҡж №жҚ®зү№ж®ҠеңәжҷҜдёҡеҠЎзҰҒжӯўжҢҮд»ӨйҮҚжҺ’д»ҺиҖҢдҝқиҜҒзЁӢеәҸжү§иЎҢзҡ„жңүеәҸжҖ§==

4. JMMи§„иҢғдёӢпјҢеӨҡзәҝзЁӢеҜ№еҸҳйҮҸзҡ„иҜ»еҶҷиҝҮзЁӢ

иҜ»еҸ–иҝҮзЁӢпјҡ
з”ұдәҺJVMиҝҗиЎҢзЁӢеәҸзҡ„е®һдҪ“жҳҜзәҝзЁӢпјҢиҖҢжҜҸдёӘзәҝзЁӢеҲӣе»әж—¶JVMйғҪдјҡдёәе…¶еҲӣе»әдёҖдёӘе·ҘдҪңеҶ…еӯҳ(жңүдәӣең°ж–№з§°дёәж Ҳз©әй—ҙ)пјҢе·ҘдҪңеҶ…еӯҳжҳҜжҜҸдёӘзәҝзЁӢзҡ„з§Ғжңүж•°жҚ®еҢәеҹҹпјҢиҖҢJavaеҶ…еӯҳжЁЎеһӢдёӯи§„е®ҡжүҖжңүеҸҳйҮҸйғҪеӯҳеӮЁеңЁдё»еҶ…еӯҳпјҢдё»еҶ…еӯҳжҳҜе…ұдә«еҶ…еӯҳеҢәеҹҹпјҢжүҖжңүзәҝзЁӢйғҪеҸҜд»Ҙи®ҝй—®пјҢ==дҪҶзәҝзЁӢеҜ№еҸҳйҮҸзҡ„ж“ҚдҪң(иҜ»еҸ–иөӢеҖјзӯү)еҝ…йЎ»еңЁе·ҘдҪңеҶ…еӯҳдёӯиҝӣиЎҢпјҢйҰ–е…ҲиҰҒе°ҶеҸҳйҮҸд»Һдё»еҶ…еӯҳжӢ·иҙқеҲ°зҡ„зәҝзЁӢиҮӘе·ұзҡ„е·ҘдҪңеҶ…еӯҳз©әй—ҙпјҢз„¶еҗҺеҜ№еҸҳйҮҸиҝӣиЎҢж“ҚдҪңпјҢж“ҚдҪңе®ҢжҲҗеҗҺеҶҚе°ҶеҸҳйҮҸеҶҷеӣһдё»еҶ…еӯҳ==пјҢдёҚиғҪзӣҙжҺҘж“ҚдҪңдё»еҶ…еӯҳдёӯзҡ„еҸҳйҮҸпјҢеҗ„дёӘзәҝзЁӢдёӯзҡ„е·ҘдҪңеҶ…еӯҳдёӯеӯҳеӮЁзқҖдё»еҶ…еӯҳдёӯзҡ„еҸҳйҮҸеүҜжң¬жӢ·иҙқпјҢеӣ жӯӨдёҚеҗҢзҡ„зәҝзЁӢй—ҙж— жі•и®ҝй—®еҜ№ж–№зҡ„е·ҘдҪңеҶ…еӯҳпјҢзәҝзЁӢй—ҙзҡ„йҖҡдҝЎ(дј еҖј)еҝ…йЎ»йҖҡиҝҮдё»еҶ…еӯҳжқҘе®ҢжҲҗпјҢе…¶з®ҖиҰҒи®ҝй—®иҝҮзЁӢеҰӮдёӢеӣҫ:

жҖ»з»“:

  • жҲ‘д»¬е®ҡд№үзҡ„жүҖжңүе…ұдә«еҸҳйҮҸйғҪеӮЁеӯҳеңЁзү©зҗҶдё»еҶ…еӯҳдёӯ
  • жҜҸдёӘзәҝзЁӢйғҪжңүиҮӘе·ұзӢ¬з«Ӣзҡ„е·ҘдҪңеҶ…еӯҳпјҢйҮҢйқўдҝқеӯҳиҜҘзәҝзЁӢдҪҝз”ЁеҲ°зҡ„еҸҳйҮҸзҡ„еүҜжң¬(дё»еҶ…еӯҳдёӯиҜҘеҸҳйҮҸзҡ„дёҖд»ҪжӢ·иҙқ)
  • зәҝзЁӢеҜ№е…ұдә«еҸҳйҮҸжүҖжңүзҡ„ж“ҚдҪңйғҪеҝ…йЎ»е…ҲеңЁзәҝзЁӢиҮӘе·ұзҡ„е·ҘдҪңеҶ…еӯҳдёӯиҝӣиЎҢеҗҺеҶҷеӣһдё»еҶ…еӯҳпјҢдёҚиғҪзӣҙжҺҘд»Һдё»еҶ…еӯҳдёӯиҜ»еҶҷ(дёҚиғҪи¶Ҡзә§)
  • дёҚеҗҢзәҝзЁӢд№Ӣй—ҙд№ҹж— жі•зӣҙжҺҘи®ҝй—®е…¶д»–зәҝзЁӢзҡ„е·ҘдҪңеҶ…еӯҳдёӯзҡ„еҸҳйҮҸпјҢзәҝзЁӢй—ҙеҸҳйҮҸеҖјзҡ„дј йҖ’йңҖиҰҒйҖҡиҝҮдё»еҶ…еӯҳжқҘиҝӣиЎҢ**(еҗҢзә§дёҚиғҪзӣёдә’и®ҝй—®)**

5. JMMи§„иҢғдёӢпјҢеӨҡзәҝзЁӢе…ҲиЎҢеҸ‘з”ҹеҺҹеҲҷд№Ӣhappens-before

еңЁJMMдёӯпјҢеҰӮжһңдёҖдёӘж“ҚдҪңжү§иЎҢзҡ„з»“жһңйңҖиҰҒеҜ№еҸҰдёҖдёӘж“ҚдҪңеҸҜи§ҒжҖ§жҲ–иҖ…д»Јз ҒйҮҚжҺ’еәҸйӮЈд№ҲиҝҷдёӨдёӘж“ҚдҪңд№Ӣй—ҙеҝ…йЎ»еӯҳеңЁhappens-beforeе…ізі»гҖӮ

5.0 е…ҲиЎҢеҸ‘з”ҹеҺҹеҲҷдҪңз”Ё

  • еҰӮжһңJavaеҶ…еӯҳжЁЎеһӢдёӯжүҖжңүзҡ„жңүеәҸжҖ§йғҪд»…йқ volatileе’ҢsynchronizedжқҘе®ҢжҲҗпјҢйӮЈд№ҲжңүеҫҲеӨҡж“ҚдҪңйғҪе°ҶдјҡеҸҳеҫ—йқһеёёе•°е—ҰпјҢдҪҶжҳҜжҲ‘д»¬еңЁзј–еҶҷJavaе№¶еҸ‘д»Јз Ғзҡ„ж—¶еҖҷе№¶жІЎжңүеҜҹи§үеҲ°иҝҷдёҖзӮ№гҖӮ

  • жҲ‘д»¬жІЎжңүж—¶ж—¶гҖҒеӨ„еӨ„гҖҒж¬Ўж¬ЎпјҢж·»еҠ volatileе’ҢsynchronizedжқҘе®ҢжҲҗзЁӢеәҸпјҢиҝҷжҳҜеӣ дёәJavaиҜӯиЁҖдёӯJMMеҺҹеҲҷдёӢжңүдёҖдёӘвҖңе…ҲиЎҢеҸ‘з”ҹвҖқ(Happens-Before)зҡ„еҺҹеҲҷйҷҗеҲ¶е’Ңи§„зҹ©

  • е®ғжҳҜеҲӨж–ӯж•°жҚ®жҳҜеҗҰеӯҳеңЁз«һдәүпјҢзәҝзЁӢжҳҜеҗҰе®үе…Ёзҡ„йқһеёёжңүз”Ёзҡ„жүӢж®өгҖӮдҫқиө–иҝҷдёӘеҺҹеҲҷпјҢжҲ‘д»¬еҸҜд»ҘйҖҡиҝҮеҮ жқЎз®ҖеҚ•и§„еҲҷдёҖжҸҪеӯҗи§ЈеҶіе№¶еҸ‘зҺҜеўғдёӢдёӨдёӘж“ҚдҪңд№Ӣй—ҙжҳҜеҗҰеҸҜиғҪеӯҳеңЁеҶІзӘҒзҡ„жүҖжңүй—®йўҳпјҢиҖҢдёҚйңҖиҰҒйҷ·е…ҘJavaеҶ…еӯҳжЁЎеһӢиӢҰж¶©йҡҫжҮӮзҡ„еә•еұӮзј–иҜ‘еҺҹзҗҶд№ӢдёӯгҖӮ

5.1 зӨәдҫӢ1

5.2. happens-beforeжҖ»еҺҹеҲҷ(йқўиҜ•еӣһзӯ”)

  1. еҰӮжһңдёҖдёӘж“ҚдҪңhappens-beforeеҸҰдёҖдёӘж“ҚдҪңпјҢйӮЈд№Ҳз¬¬дёҖдёӘж“ҚдҪңзҡ„жү§иЎҢз»“жһңе°ҶеҜ№з¬¬дәҢдёӘж“ҚдҪңеҸҜи§ҒпјҢиҖҢдё”з¬¬дёҖдёӘж“ҚдҪңзҡ„жү§иЎҢйЎәеәҸжҺ’еңЁз¬¬дәҢдёӘж“ҚдҪңд№ӢеүҚгҖӮ(иҝҷдёҚе°ұжҳҜдҫқиө–жҖ§еҗ—?еҜ№жҹҗдёӘж“ҚдҪңжңүдҫқиө–,иӮҜе®ҡе…Ҳжү§иЎҢдҫқиө–е•Ҡ)
  2. дёӨдёӘж“ҚдҪңд№Ӣй—ҙеӯҳеңЁhappens-beforeе…ізі»пјҢе№¶дёҚж„Ҹе‘ізқҖдёҖе®ҡиҰҒжҢүз…§happens-beforeеҺҹеҲҷеҲ¶е®ҡзҡ„йЎәеәҸжқҘжү§иЎҢгҖӮеҰӮжһңйҮҚжҺ’еәҸд№ӢеҗҺзҡ„жү§иЎҢз»“жһңдёҺжҢүз…§happens-beforeе…ізі»жқҘжү§иЎҢзҡ„з»“жһңдёҖиҮҙпјҢйӮЈд№Ҳиҝҷз§ҚйҮҚжҺ’еәҸе№¶дёҚйқһжі•гҖӮ

5.3. happens-beforeд№Ӣ8жқЎ

5.3.1 ж¬ЎеәҸи§„еҲҷ

дёҖдёӘзәҝзЁӢеҶ…пјҢжҢүз…§д»Јз ҒйЎәеәҸпјҢеҶҷеңЁеүҚйқўзҡ„ж“ҚдҪңе…ҲиЎҢеҸ‘з”ҹдәҺеҶҷеңЁеҗҺйқўзҡ„ж“ҚдҪңпјӣ(еүҚдёҖдёӘж“ҚдҪңзҡ„з»“жһңеҸҜд»Ҙиў«еҗҺз»ӯзҡ„ж“ҚдҪңиҺ·еҸ–гҖӮ)

5.3.2 й”Ғе®ҡи§„еҲҷ

дёҖдёӘunLockж“ҚдҪңе…ҲиЎҢеҸ‘з”ҹдәҺеҗҺйқў((иҝҷйҮҢзҡ„вҖңеҗҺйқўвҖқжҳҜжҢҮж—¶й—ҙдёҠзҡ„е…ҲеҗҺ))еҜ№еҗҢдёҖдёӘй”Ғзҡ„lockж“ҚдҪңпјӣ(еҜ№дәҺеҗҢдёҖжҠҠй”ҒobjectLockпјҢthreadAдёҖе®ҡе…ҲunlockеҗҢдёҖжҠҠй”ҒеҗҺBжүҚиғҪиҺ·еҫ—иҜҘй”ҒпјҢ A е…ҲиЎҢеҸ‘з”ҹдәҺB)

5.3.3 volatileеҸҳйҮҸи§„еҲҷ

еҜ№дёҖдёӘvolatileеҸҳйҮҸзҡ„еҶҷж“ҚдҪңе…ҲиЎҢеҸ‘з”ҹдәҺеҗҺйқўеҜ№иҝҷдёӘеҸҳйҮҸзҡ„иҜ»ж“ҚдҪңпјҢеүҚйқўзҡ„еҶҷеҜ№еҗҺйқўзҡ„иҜ»жҳҜеҸҜи§Ғзҡ„пјҢиҝҷйҮҢзҡ„вҖңеҗҺйқўвҖқеҗҢж ·жҳҜжҢҮж—¶й—ҙдёҠзҡ„е…ҲеҗҺгҖӮ

5.3.4 дј йҖ’и§„еҲҷ

еҰӮжһңж“ҚдҪңAе…ҲиЎҢеҸ‘з”ҹдәҺж“ҚдҪңBпјҢиҖҢж“ҚдҪңBеҸҲе…ҲиЎҢеҸ‘з”ҹдәҺж“ҚдҪңCпјҢеҲҷеҸҜд»Ҙеҫ—еҮәж“ҚдҪңAе…ҲиЎҢеҸ‘з”ҹдәҺж“ҚдҪңCпјӣ

5.3.5 зәҝзЁӢеҗҜеҠЁи§„еҲҷ(Thread Start Rule)

ThreadеҜ№иұЎзҡ„start()ж–№жі•е…ҲиЎҢеҸ‘з”ҹдәҺжӯӨзәҝзЁӢзҡ„жҜҸдёҖдёӘеҠЁдҪң

5.3.6 зәҝзЁӢдёӯж–ӯи§„еҲҷ(Thread Interruption Rule)

еҜ№зәҝзЁӢinterrupt()ж–№жі•зҡ„и°ғз”Ёе…ҲиЎҢеҸ‘з”ҹдәҺиў«дёӯж–ӯзәҝзЁӢзҡ„д»Јз ҒжЈҖжөӢеҲ°дёӯж–ӯдәӢд»¶зҡ„еҸ‘з”ҹпјӣ

еҸҜд»ҘйҖҡиҝҮThread.interrupted()жЈҖжөӢеҲ°жҳҜеҗҰеҸ‘з”ҹдёӯж–ӯ

5.3.7 зәҝзЁӢз»Ҳжӯўи§„еҲҷ(Thread Termination Rule)

зәҝзЁӢдёӯзҡ„жүҖжңүж“ҚдҪңйғҪе…ҲиЎҢеҸ‘з”ҹдәҺеҜ№жӯӨзәҝзЁӢзҡ„з»ҲжӯўжЈҖжөӢпјҢ

жҲ‘д»¬еҸҜд»ҘйҖҡиҝҮThread::join()ж–№жі•жҳҜеҗҰз»“жқҹгҖҒThread::isAlive()зҡ„иҝ”еӣһеҖјзӯүжүӢж®өжЈҖжөӢзәҝзЁӢжҳҜеҗҰе·Із»Ҹз»Ҳжӯўжү§иЎҢгҖӮ

5.3.8 еҜ№иұЎз»Ҳз»“и§„еҲҷ(Finalizer Rule)

дёҖдёӘеҜ№иұЎзҡ„еҲқе§ӢеҢ–е®ҢжҲҗпјҲжһ„йҖ еҮҪж•°жү§иЎҢз»“жқҹпјүе…ҲиЎҢеҸ‘з”ҹдәҺе®ғзҡ„finalize()ж–№жі•зҡ„ејҖе§Ӣ

(еҜ№иұЎжІЎжңүе®ҢжҲҗеҲқе§ӢеҢ–д№ӢеүҚпјҢжҳҜдёҚиғҪи°ғз”Ёfinalized()ж–№жі•зҡ„)

5.4 зӨәдҫӢ2

5.4.1 дјҳеҢ–ж–№ејҸ1:жҠҠgetter/setterж–№жі•йғҪе®ҡд№үдёәsynchronizedж–№жі•

иҷҪз„¶и§ЈеҶідәҶзәҝзЁӢе®үе…Ёзҡ„й—®йўҳ,дҪҶжҳҜиҜ»е’ҢеҶҷзҡ„е№¶еҸ‘йҮҸйғҪдёӢйҷҚдәҶ

5.4.2 (жҺЁиҚҗ)дјҳеҢ–ж–№ејҸ2:жҠҠvalueе®ҡд№үдёәvolatileеҸҳйҮҸпјҢз”ұдәҺsetterж–№жі•еҜ№valueзҡ„дҝ®ж”№дёҚдҫқиө–valueзҡ„еҺҹеҖјпјҢж»Ўи¶іvolatileе…ій”®еӯ—дҪҝз”ЁеңәжҷҜ

5.5 happens-beforeжҖ»з»“

  • еңЁJava иҜӯиЁҖйҮҢйқўпјҢHappens-Before зҡ„иҜӯд№ү==жң¬иҙЁдёҠжҳҜдёҖз§ҚеҸҜи§ҒжҖ§==
  • A Happens-Before Bж„Ҹе‘ізқҖAеҸ‘з”ҹиҝҮзҡ„дәӢжғ…еҜ№BжқҘиҜҙжҳҜеҸҜи§Ғзҡ„пјҢж— и®әAдәӢд»¶е’ҢBдәӢд»¶жҳҜеҗҰеҸ‘з”ҹеңЁеҗҢдёҖдёӘзәҝзЁӢйҮҢ.
  • JMMзҡ„и®ҫи®ЎеҲҶдёәдёӨйғЁеҲҶ:
    дёҖйғЁеҲҶжҳҜйқўеҗ‘жҲ‘д»¬зЁӢеәҸе‘ҳжҸҗдҫӣзҡ„пјҢд№ҹе°ұжҳҜhappens-beforeи§„еҲҷпјҢе®ғйҖҡдҝ—жҳ“жҮӮзҡ„еҗ‘жҲ‘д»¬зЁӢеәҸе‘ҳйҳҗиҝ°дәҶдёҖдёӘејәеҶ…еӯҳжЁЎеһӢпјҢжҲ‘д»¬еҸӘиҰҒзҗҶи§Јhappens-beforeи§„еҲҷпјҢе°ұеҸҜд»Ҙзј–еҶҷе№¶еҸ‘е®үе…Ёзҡ„зЁӢеәҸдәҶгҖӮ
  • еҸҰдёҖйғЁеҲҶжҳҜй’ҲеҜ№JVMе®һзҺ°зҡ„пјҢдёәдәҶе°ҪеҸҜиғҪе°‘зҡ„еҜ№зј–иҜ‘еҷЁе’ҢеӨ„зҗҶеҷЁеҒҡзәҰжқҹд»ҺиҖҢжҸҗй«ҳжҖ§иғҪпјҢJMMеңЁдёҚеҪұе“ҚзЁӢеәҸжү§иЎҢз»“жһңзҡ„еүҚжҸҗдёӢеҜ№е…¶дёҚеҒҡиҰҒжұӮпјҢеҚіе…Ғи®ёдјҳеҢ–йҮҚжҺ’еәҸгҖӮжҲ‘д»¬еҸӘйңҖиҰҒе…іжіЁеүҚиҖ…е°ұеҘҪдәҶ,д№ҹе°ұжҳҜзҗҶи§Јhappens-beforeи§„еҲҷеҚіеҸҜпјҢе…¶е®ғз№ҒжқӮзҡ„еҶ…е®№жңүJMMи§„иҢғз»“еҗҲж“ҚдҪңзі»з»ҹз»ҷжҲ‘д»¬жҗһе®ҡпјҢжҲ‘д»¬еҸӘеҶҷеҘҪд»Јз ҒеҚіеҸҜгҖӮ

6. йқўиҜ•йўҳ

  1. дҪ зҹҘйҒ“д»Җд№ҲжҳҜJavaеҶ…еӯҳжЁЎеһӢJMMеҗ—пјҹ
  2. JMMдёҺvolatileе®ғд»¬дёӨдёӘд№Ӣй—ҙзҡ„е…ізі»пјҹ
  3. JMMжңүе“Әдәӣзү№жҖ§orе®ғзҡ„дёүеӨ§зү№жҖ§жҳҜд»Җд№Ҳпјҹ
  4. дёәд»Җд№ҲиҰҒжңүJMMпјҢе®ғдёәд»Җд№ҲеҮәзҺ°пјҹдҪңз”Ёе’ҢеҠҹиғҪжҳҜд»Җд№Ҳпјҹ
  5. happens-beforeе…ҲиЎҢеҸ‘з”ҹеҺҹеҲҷдҪ жңүдәҶи§ЈиҝҮеҗ—пјҹ
