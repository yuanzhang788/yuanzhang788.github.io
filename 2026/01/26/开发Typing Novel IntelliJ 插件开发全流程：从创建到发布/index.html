发表于2026-01-26|更新于2026-06-26

如果只是想单纯体验一下可以选择Say-Hello分支(即本文内容),主分支用于本人Typing Novel 插件的开发(2025/12/19 正在开发中)

一、为什么要写这个插件？

最近在做一个 Typing Novel（打字联动小说展示） 的 IDEA 插件，在正式做功能前，第一步一定是：

让插件“活起来” —— 在 IDEA 菜单里点一下，弹个 Hello World

看似简单，但在 IDEA 2025+ 之后，插件体系发生了比较大的变化：

  • gradle-intellij-plugin 1.x已不支持 2024.2+
  • 必须使用 IntelliJ Platform Gradle Plugin 2.x
  • Java / Kotlin 混用有隐藏坑

所以有必要把整个过程完整记录下来。

二、项目创建与技术栈选择

技术选型

项目 选择
IntelliJ Platform 2025.1.4.1
Gradle 插件 org.jetbrains.intellij.platform
JDK 21
语言 Kotlin（支持 Java，但需注意坑）

安装 Plugin DevKit 插件

从 IntelliJ IDEA 2022.2 版本开始，JetBrains 不再默认安装 Plugin DevKit 插件。需要手动从插件市场安装该插件才能进行 IntelliJ 平台插件开发。

创建项目

新建项目，点击 File -> New -> Projects。

选择 IDE Plugin 项目，然后在右侧配置自己的项目信息。

如果弹出以下错误就是,需要JDK21 ,最好安装oracle的,我当时图方便直接在IDEA下载了微软的JDK21,后面发现是不兼容的

安装好jdk后创建项目

项目创建完成后来到build.gradle.kts

点击Sync Gradle Change 会下载intellijPlatform相关依赖

三、编写对应功能

完成后就可以开始开发了,因为我平常使用的JAVA,所以我还是选用JAVA进行开发,这里可以和kotlin二选一看读者熟悉哪个,默认是没有java这个目录的需要自己创建,结构如下

HelloAction.java

通过plugin.xml把 HelloAction 类“注册”到 IDEA 的 UI 系统里

配置resources-> META-INF -> plugin.xml

这段 <actions> 的作用是：
👉 把 HelloAction 类“注册”到 IDEA 的 UI 系统里，并决定它出现在什么菜单、什么位置。

  • class: 为我们编写的HelloAction类的全路径类名
  • text: 为我们这个行为的操作的名称,后续执行成功可以看到
  • group-id: IDEA 所有菜单、工具栏、本质上都是 **ActionGroup ,**如下面的表格所示,我选用的是ToolsMenu,所以就是在Tools 菜单增加操作
  • anchor=”last”: 表示 表示放到 菜单最后
group-id 对应位置
ToolsMenu Tools 菜单
MainMenu 顶部主菜单
EditorPopupMenu 编辑器右键菜单
ProjectViewPopupMenu 项目树右键

点击过后执行HelloAction的操作弹出一个对应提示框

五、上传插件

开始上传插件

  1. 找到上传入口
    1. Idea 插件市场地址：https://plugins.jetbrains.com/，没有账号的话可以用邮箱注册下。也可以用谷歌账号直接登录

  1. 同意相关协议

  1. 开始上传插件,填写相关信息

  1. 上传插件

  1. 点击Upload Plugin 后会进行插件检测,如果出现以下报错就是检查不通过

这些报错 不是构建失败，而是 JetBrains 官方的插件规范校验，我们逐条解决即可。

解决规范校验问题（重点）

❌ 1. 插件名不能包含 plugin

报错

原因

JetBrains 明确规定：

插件名里 不要出现 plugin / IDE / IntelliJ

❌ 2. description 还是默认模板内容

报错

📌 支持 HTML，但**不要留默认文本,**填写对应介绍即可

❌ 3. vendor 还是 YourCompany

报错

可以是：

  • 个人名
  • GitHub 名
  • 团队名

❌ 4. vendor url 还是模板地址

报错

修改为自己的github或者gitee 地址即可

然后重新打包上传,上传成功

上传成功后点击Open Plugin Page 就可以进到插件主页了,但是这个时候在Idea 插件商店搜不到,还需要审核通过才能搜索到

IDEA 审核大概需要两个工作日的时间，如果审核没有问题，就能搜索到自己的插件了
