7.map 和 reduce

数组中新增了 map 和 reduce 方法 .

1.map方法

map()： 接收一个函数， 将原数组中的所有元素用这个函数处理后放入新数组返回。

结果:对数组中的所有元素乘2操作并且返回一个新的数组

2.reduce方法

reduce() 为数组中的每一个元素依次执行回调函数，不包括数组中被删除或从未被赋值的元素，

语法:arr.reduce(callback,[initialValue])

  • callback （执行数组中每个值的函数， 包含四个参数）
  1. previousValue （上一次调用回调返回的值， 或者是提供的初始值（initialValue） ）
  2. currentValue （数组中当前被处理的元素）
  3. index （当前元素在数组中的索引）
  4. array （调用 reduce 的数组）
  • initialValue （作为第一次调用 callback 的第一个参数。 ）

结果:有四个元素,循环4次.

  • 第一次:初始值我们赋值100,所以上一次处理后的值为100,正在处理第一个元素2,得到result102
  • 第二次:上一次处理后的值为102,正在处理第二个元素40,得到result142
  • 第三次:上一次处理后的值为142,正在处理第三个元素-10,得到result132
  • 第四次:上一次处理后的值为132,正在处理第四个元素6,得到result138,得到最终结果138并返回

9.模块化

1.什么是模块化

模块化就是把代码进行拆分， 方便重复利用。 类似 java 中的导包： 要使用一个包， 必须先导包。 而 JS 中没有包的概念， 换来的是 模块。
模块功能主要由两个命令构成： export和import。

  • export命令用于规定模块的对外接口。
  • import命令用于导入其他模块提供的功能。

2.export&import

export不仅可以导出对象， 一切 JS 变量都可以导出。 比如： 基本类型变量、 函数、 数组、对象。

对象导出:

  • 普通写法
  • 明确指定了导出的变量名， 这样其它人在导入使用时就必须准确写出变量名， 否则就会出错。
  • 简写
  • js 提供了default关键字， 可以对导出的变量名进行省略
