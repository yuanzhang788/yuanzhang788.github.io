  • Redis 的 SCAN 命令用于迭代数据库中的键，支持通过模式过滤结果。模式规则基于 Redis 的通配符匹配语法，类似于文件名匹配规则：
    • *：匹配零个或多个字符。
    • ?：匹配一个字符。
    • [abc]：匹配括号中的任意一个字符。
    • [a-z]：匹配括号中的字符范围。
    • 如:ScanOptions.scanOptions().match(“*favorites:”)即为匹配以favorites:结尾的key
